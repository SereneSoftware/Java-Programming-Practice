# Java-Programming-Practice
💻 Collection of beginner-friendly Java programs covering conditionals, loops, operators, and basic logic building.
## Topics Covered

- Arithmetic operators
- Conditional statements (if-else, switch)
- Loops
- Input handling
- Basic logic building

## Programs Included

- Simple Calculator
- Guess the Number
- Coffee Order System
- College Selector
- Ticket Booking
- Password Validation
- Discount Calculator
- Fruit Sale System
- Movie Selector
- Subject Eligibility Check
- Sum and Average Calculator
- Days in a Week Program
- Increment & Decrement Examples
- Divisibility Check
- Juice Shop Billing

## Purpose

These programs were developed as practice exercises to improve problem-solving skills and build a strong foundation in Java.
